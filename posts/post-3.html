10 concepts and examples of DAX functions that are widely used in my daily

Part 3

Published by Daniel Fernandes on January 20th, 2024

Introduction

In this third post of the series "Concepts and examples of DAX functions widely used in everyday life", I present 10 text functions that I encounter daily. These functions work with tables and columns in table models. Here are the examples:

1. COMBINEVALUES:

Purpose: Combines two values into a single string.

      
       TotalSalesExceptProduct =
       CALCULATE(
          SUM('Sales'[Value]),
          ALLEXCEPT('Sales', 'Products'[Name])
       )

2. CONCATENATE:

Purpose: Concatenates two or more strings.

      
         ConcatenatedString = CONCATENATE('Table'[FirstName], " ", 'Table'[LastName])

3. CONCATENATEX:

Purpose: Concatenates values from a column in a table using an expression.

      
         ConcatenatedValues = CONCATENATEX('Table', 'Table'[Column], ", ")

4. EXACT:

Purpose: Returns TRUE if two texts are identical, otherwise, returns FALSE.

      
         IsExactMatch = EXACT('Table'[Text1], 'Table'[Text2])

5. FIND:

Purpose: Returns the position of one text within another text.

      
         Position = FIND("search", 'Table'[Text])

6. FIXED:

Purpose: Returns a number formatted as text with a fixed number of digits.

      
         FormattedNumber = FIXED('Table'[Number], 2, 1)

7. FORMAT:

Purpose: Formats a value based on a provided format.

      
         FormattedDate = FORMAT('Table'[Date], "dd-mmm-yyyy")

8. LEFT:

Purpose: Returns the leftmost characters of a string.

      
         LeftSubstring = LEFT('Table'[Text], 5)

9. LEN:

Purpose: Returns the length of a string.

      
         StringLength = LEN('Table'[Text])

10. LOWER:

Purpose: Converts a string to lowercase.

      
         LowercaseText = LOWER('Table'[Text])
